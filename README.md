# StockAlert Trading Analytics Cloud

## Overview

StockAlert Trading Analytics Cloud is a cloud-based stock market analytics and monitoring platform developed using AWS services and modern web technologies. The platform provides centralized visibility into stock data, reporting, monitoring, and cloud infrastructure management.

## Technology Stack

### Frontend

* React.js
* Vite
* Axios
* React Router

### Backend

* Node.js
* Express.js

### Database

* MySQL (AWS RDS)

### Cloud Services

* AWS EC2
* AWS RDS
* AWS S3
* AWS CloudWatch
* AWS Security Groups

### DevOps

* Docker
* GitHub

## Features

### Login Module

* User authentication interface
* Secure access to application resources

### Dashboard Module

* Centralized operational dashboard
* KPI and analytics overview

### Stocks Module

* Stock data management
* API-driven stock information display

### Reports Module

* Reporting and analytics dashboard
* Business insights visualization

## Deployment Architecture

User Browser → React Frontend → Express Backend (EC2) → AWS RDS MySQL

Additional Services:

* AWS S3 for storage
* AWS CloudWatch for monitoring
* Docker for containerization

## AWS Services Used

* EC2 for application hosting
* RDS MySQL for database management
* S3 for storage and reporting
* CloudWatch for monitoring
* Security Groups for network access control

## Deployment Workflow

Local Development → GitHub Repository → AWS EC2 Deployment → Public Access
